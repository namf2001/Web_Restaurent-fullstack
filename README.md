# Create App
In the project directory, you can run:
## *backend*
### `cd ./backend && npm install or yarn install`
### `npm start or yarn start`

## *frontend*
### `cd ./frontend && npm install or yarn install`
### `npm dev or yarn dev`

# Tech Stack
    - React
    - Node
    - Express
    - MongoDB
    - Tailwind
    - Swiper
    - ...
